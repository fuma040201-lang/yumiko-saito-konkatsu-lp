# LINE導線案｜齋藤弓子 結婚相談所オーナー向けLP

## 目的

LP閲覧者を単なる友だち追加で終わらせず、現在地の診断、最初の行動、必要な人だけ個別相談へつなぐ。

## 登録オファー

- 名称：相談所集客 5分診断
- 登録特典：14日間実践ロードマップ
- 対象：会員10名以下、特に0〜3名で止まっている結婚相談所
- 約束：年商億越え相談所で結果が出た順番を、14日間そのまま実行できる
- 注意：会員獲得や売上を保証する表現は使わない

## 登録直後の診断

1. 現在の会員数：0名／1〜3名／4〜10名／11名以上
2. 直近30日の問い合わせ：0件／1〜2件／3件以上
3. 現在の集客方法：ホームページ／SNS／紹介／広告／まだ決めていない
4. 1週間に集客へ使える時間：1時間未満／1〜3時間／4時間以上
5. 一番の悩み：見込み客が来ない／問い合わせにつながらない／何をすべきか分からない

## 診断結果の3タイプ

### A｜接点不足タイプ

見込み客に存在を知ってもらう接点が少ない。14日間の前半で、商圏内の接点を一つずつ増やす。

### B｜伝わり方不足タイプ

発信やホームページはあるが、誰のどんな悩みを解決する相談所か分かりにくい。14日間の最初に、訴求を一つに絞る。

### C｜行動分散タイプ

複数施策を同時に進め、反応の良し悪しが判断できない。14日間は一つの手順に絞り、反応の数字を見る。

## 配信シナリオ

- 登録直後：お礼 → 5分診断スタート
- 診断完了直後：タイプ結果 → 14日間実践ロードマップ
- 1日後：よくある失敗と、今日15分でできる一手
- 3日後：14日間の進め方を短く紹介
- 5日後：実行できたか確認 → 希望者だけ60分個別相談

## 個別相談への接続

- 診断結果を引き継いだ状態で申し込める
- 取得項目：相談所名、氏名、希望日時、現在一番困っていること
- 相談内容：現状整理、最初の施策決定、14日間の確認指標
- 「無理な勧誘なし」「相談のみ可」を明記

## 実装前の確認事項

- 齋藤弓子さんのLINE公式アカウントURLとBasic ID
- 使用ツール：Lステップ、エルメ、LINE Harnessなど
- 既存のあいさつメッセージ、タグ、リッチメニュー、予約導線
- 実績表記の証憑と公開可能範囲
- 配信を有効化する前に、テストアカウントで診断分岐と予約通知を確認する
